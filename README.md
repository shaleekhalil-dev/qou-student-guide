# الدليل الإرشادي الرقمي الموحد لطلبة الماجستير الجدد - جامعة القدس المفتوحة

## وصف المشروع
منصة إلكترونية تفاعلية مخصصة لمساعدة وتوجيه طلبة الدراسات العليا الجدد في تخصص إدارة الموارد البشرية التطبيقية بجامعة القدس المفتوحة. يهدف المشروع إلى معالجة وتذليل كافة العقبات الإجرائية، التقنية، والمالية التي تواجه الطلبة في مرحلة الالتحاق عبر مسار رقمي موحد يركز على تحسين تجربة المستخدم واختصار التشتت الإجرائي والجغرافي.

[cite_start]تم بناء هذا الموقع كدراسة حالة تطبيقية متكاملة لربط الحلول البرمجية بالاحتياجات الفعلية للطلاب بالاستناد المباشر إلى منهجية التفكير التصميمي[cite: 39].

## الهوية البصرية ونمط التصميم
يعتمد الموقع في بنية واجهاته بالكامل على نمط **التصميم الزجاجي المتدرج الأخضر الفاتح جداً (Light Green Glassmorphism)**، والذي يدمج بين حداثة المظهر الانسيابي ووضوح العناصر البصرية والمحاذاة الكاملة لليمين، مما يضمن تصفحاً مريحاً يمنع تشتت الطلبة أثناء استقراء المعلومات.

## هيكلية المنصة (7 صفحات أساسية)
1. **الصفحة الرئيسية (index.html):** ترحيب بالطلبة الجدد، تعريف برؤية عمادة الدراسات العليا، استعراض تخصصات الماجستير المتاحة، ووصف مفصل لمحاور الدليل.
2. **إجراءات التسجيل (registration.html):** تسلسل خطوات القبول والالتحاق، حصر الوثائق والمستندات الثبوتية المطلوبة، طرق الحصول على رسائل التوصية الأكاديمية، وجدول تفصيلي بالرسوم الدراسية وحجز المقاعد.
3. **البوابة والتيمز (portal.html):** إرشاد الدخول الأولي للبوابة الأكاديمية الموحدة، عرض الرسالة التوجيهية المنبثقة وتحديث البيانات، شرح مكونات لوحة التحكم، وخطوات تفعيل حساب Microsoft Teams عن بُعد.
4. **المساقات والجداول (courses.html):** تفصيل هيكلية الساعات المعتمدة للتخرج، الجداول والخطط الدراسية المقترحة لطلبة السنة الأولى والسنة الثانية فصلياً، ضوابط تسجيل مواد الفصل الصيفي، وآلية التسجيل الإلكتروني.
5. **متطلبات الدراسة (requirements.html):** توضيح طبيعة الواجبات والتكليفات الفصلية المفروضة (الامتحانات النصفية والنهائية، الأوراق البحثية، التقارير الأسبوعية)، وشروط الجاهزية لإعداد وعرض المواد التعليمية (Presentations).
6. **الأسئلة والمراجع (faqs.html):** حلول وإجابات مباشرة لأكثر المشكلات التقنية والمالية شيوعاً، روابط الوصول السريع لأقسام الخدمات داخل البوابة، والدليل المنهجي المعتمد لتصميم الأوراق العلمية والبحوث وفق نظام التوثيق العالمي (APA Style).
7. **عن المشروع (about.html):** تعريف شامل بمراحل التفكير التصميمي الخمس، وتوثيق الإشراف الأكاديمي وتوزيع أدوار ومسؤوليات فريق العمل.
